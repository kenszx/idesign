## 四、27 动画速查

**路径**: `assets/animations/animations.css`（CSS 动画）
**路径**: `assets/animations/fx/*.js`（Canvas 粒子效果）

| 动画 | 类型 | 用法 |
|------|------|------|
| `anim-fade-up` | CSS | `class="anim-fade-up"` |
| `anim-fade-in` | CSS | `class="anim-fade-in"` |
| `anim-rise-in` | CSS | `class="anim-rise-in"` |
| `anim-scale-in` | CSS | `class="anim-scale-in"` |
| `anim-slide-left` | CSS | `class="anim-slide-left"` |
| `data-anim` 属性触发 | 27+ | `data-anim="fade-up"` 等 |
| `data-fx` 粒子效果 | 20 | `data-fx="particle-burst"` 等 |

**用法**：
```html
<!-- CSS class -->
<h1 class="anim-fade-up">标题</h1>

<!-- data-anim 属性（运行时 JS 触发） -->
<div data-anim="stagger-list">...</div>

<!-- Canvas 粒子 -->
<canvas data-fx="confetti-cannon"></canvas>
```
