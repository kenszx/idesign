# Anti-Slop Rules

Shared rules for all design sub-skills that generate HTML/CSS. See `shared/references/design-principles.md` for full details.

## Quick Reference

1. **No aggressive gradient backgrounds** — Avoid purple-to-blue, sunset, conic rainbows
2. **No emoji** unless the brand uses them
3. **No rounded-corner cards with left-border accent stripe** — Most overused pattern
4. **No SVG-drawn imagery as substitute for real assets** — Use placeholders
5. **No CSS silhouettes for product shots**
6. **No decorative gradient orbs "representing AI"**
7. **No overused font stacks** — Inter/Roboto/system-ui defaults
8. **No decoration-by-dataviz** — Every number should mean something
9. **No "3 column feature grid" as default** — Consider alternatives
10. **No over-iconified bullet lists** — Plain bullets read faster
